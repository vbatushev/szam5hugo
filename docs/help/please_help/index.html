Когда ничего не получается

Такое бывает. Но ничего страшного, ведь меня можно найти по следующим адресам:

  1. Telegram Тут меня застать легче всего.
  2. Skype
  3. Электронная почта
  4. Facebook

Я часто предлагаю пользователям помощь удаленно с помощью программы TeamViewer. Она позволяет как управлять удаленным компьютером, так и общаться голосом или чатом между мной и вами, что очень удобно.

Немного полезных ссылок для тех, кто пишет правила:

  • InDesign JavaScript Reference Guide (eng.) в формате CHM и HTML
  • Помощь по Adobe InDesign, раздел “Найти/Заменить” (рус.)
  • InDesign ExtendScript API (eng): для CS6, другие версии InDesign, Illustrator и Photoshop

И тем, кто пишет скрипты сам:

  • Отличный бесплатный мультиплатформенный редактор кода Microsoft VSCode
  • Adobe ExtendScript Debugger Extension для Visual Studio Code позволяет производить отладку и компилирование скриптов в JSXBIN прямо в редакторе VSCode
  • Раздел форума RU|DTP, посвященный скриптам и плагинам для InDesign
